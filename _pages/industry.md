Here is a list of my employment experiences in the industry.

# AI/ML and Robotics Engineer at Argon Robotics [2024-Present]

Argon Robotics is a startup that aims to build a fully-autonomous welding robot manipulator for the construction industry. Some of my work has involved:

* Developing 3D reconstruction algorithms to help scan objects to be welded. Minimize noise and improve the quality of the 3D models.
* Designing deep learning models to detect the seams in the objects to be welded.
* Working with ROS2 along with the Moveit2 and Xarm libraries to control the robot manipulator.
* Designing control algorithms to help the robot manipulator move smoothly and accurately along the seams.

As you can see, this is a pretty hard task -- and pretty exciting too!

# Software Engineering Lead at the Innovative Marine Technology Lab at the Scripps Institution of Oceanography [2023-Present]

I am currently leading the software engineering work at the Innovative Marine Technology Lab at the Scripps Institution of Oceanography, and I am advised by Dr. Grant Deane and Dr. Dale Stokes. The lab works on building underwater robots that can be used for oceanographic research and environmental monitoring. Some of my work has involved:

* Building circuits used to control devices like aerosol particle counters.
* Writing software to operate underwater cameras and sensors.
* Using the Raspberry Pi to control different sensors and devices on the underwater robots.
* Developing a full-stack web application for visualizing data collected by the underwater robots and a GUI for controlling the different sensors and devices.
* Designing computer vision algorithms for detecting bubbles in underwater videos.

# Applied AI/ML Software Engineering Internship at Brain Corp [Summer 2024]

Brain Corp ([https://www.braincorp.com/](https://www.braincorp.com/)) is a San Diego-based company that develops AI-powered software for autonomous robots, primarily for tasks such as floor cleaning, inventory management, and shelf scanning. As part of the Applied AI/ML team, I trained deep learning models using Google Vertex AI to detect out-of-stock items on images of grocery store shelves. 

One of the challenges I faced was the lack of a large dataset of shelf images with out-of-stock items. I thus collected and manually annotated a dataset of shelf images from various grocery stores and managed this data using tools like Darvin V7 and BigQuery. The resulting model I trained obtained a high accuracy and was eventually deployed to improve the accuracy of out-of-stock detection.

# AI/ML and Computer Vision Engineer at NeuronFlo [2023-2024]

NeuronFlo is a startup that develops AI-powered computer vision methods for monitoring the safe use of personal protective equipment (PPE) in industrial settings to ensure worker safety. I was responsible for developing and deploying computer vision models for detecting and tracking PPE usage in real-time. I trained object detection models using PyTorch to detect different kinds of PPE, such as helmets, safety vests, and gloves. This model was then deployed on a Jetson Nano device for real-time inference and I wrote code for the downstream processing of these detections to help in monitoring and alerting the workers in case of non-compliance with safety regulations.
